# Mini User Dashboard

A fully responsive cryptocurrency mini-dashboard built using React and custom CSS, with light/dark theme toggle.

## Features
- Live crypto cards
- Responsive grid layout
- Light/Dark mode via toggle switch
- Gradient title text based on theme
- Clean UI with modern design

This guide explains how to run the backend and frontend locally for development.
